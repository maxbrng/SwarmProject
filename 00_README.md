# Art+Code Projekt — Übersicht

Arbeitsmappe für dein Uni-Projekt: ein eigenes, interaktives Kunstwerk auf Touchscreen, basierend auf einem autonomen, naturbasierten komplexen System, in das man per Geste (u.a. Swirl) eingreifen kann.

## Dateien (bewusst nur vier)

- **`01_Wissenssammlung_Komplexe_Systeme.md`** — Wissensbasis + große, recherchierte **Inspirationsgalerie**: alle Methoden (Boids, Physarum, Reaktions-Diffusion, Fluid, zelluläre Automaten, genetische Kunst, Mimesis) erklärt, plus Künstler, Werke und Links zum Reinklicken (Abschnitt 12).
- **`02_Konzeptideen.md`** — die zentrale Konzept-Datei: drei Leitprinzipien, die fünf autonomen Basissysteme (A–E) mit Story + Inspiration, die Interaktions-Ebene (Swirl + Gesten) und eine Empfehlung. **Hier wählst du.**
- **`03_Technik_Setup_und_Anforderungen.md`** — vom p5-Web-Editor zum größeren Projekt (Setup, Struktur, Git, Shader, Sound, Touchscreen-Deployment) + fertige technische Anforderungsliste fürs Miro-Board.

## Kernidee

Aus einfachen, lokalen Regeln entsteht ein komplexes, sich endlos veränderndes System. Es fesselt schon von allein; die Berührung (Swirl/Gesten) stört es nur lokal — und es heilt sich selbst. Daraus die Story: *Eingriff & Selbstheilung / Ordnung ↔ Chaos.*

## Nächster Schritt

Wähle in `02` (1) ein **Basissystem** (A Physarum / B Reaktions-Diffusion / C Boids / D Wellen / E Flussfeld) und (2) **2–3 Gesten** (z.B. Tippen=Explosion, Halten=Swirl, Ziehen=durchziehen) → dann wird der erste lauffähige p5.js-Sketch gebaut.
